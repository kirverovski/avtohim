для бесконтактной и ручной мойки
Активная пена «Эконом» (1 фаза)
100 ₽/л (2000 ₽/20 л)
Щелочной шампунь для бесконтактной мойки. pH 12–13. Для предварительной мойки и сильных загрязнений.
Активная пена «Стандарт» (1 фаза)
175 ₽/л (3500 ₽/20 л)
Густая стабильная пена, экономный расход. Подходит для ежедневной мойки легкового и коммерческого транспорта.
Активная пена «Премиум» (1 фаза)
по запросу
Усиленная формула для сложных загрязнений и детейлинга. Рекомендуется для премиальных моек.
Шампунь ручной мойки «Премиум» (2 фаза)
120 ₽/л (2400 ₽/20 л)
Разведение 1:100–1:200. Обильная пена, мягкое скольжение, бережное отношение к ЛКП.
Шампунь универсальный (2 фаза)
по запросу
Для ручной и аппаратной мойки. Подходит для моек с большим потоком автомобилей.
Для подбора оптимальной концентрации под вашу мойку — напишите в WhatsApp, подскажем по дозировкам и расходу.

290–350 ₽/л

Очиститель стекол «Премиум»
105 ₽/л
Быстро удаляет разводы, налёт и следы насекомых. Не оставляет плёнки, подходит для зеркал и хрома.
Очиститель дисков (кислотный)
150 ₽/л (3000 ₽/20 л)
Для удаления налёта, тормозной пыли и сложных загрязнений. Использовать с соблюдением техники безопасности.
Универсальный очиститель
70 ₽/л (1400 ₽/20 л)
Для пластика, резины, текстиля и подкапотного пространства. Рабочий инструмент для ежедневной мойки.
При необходимости можем добавить в линейку специализированные очистители под ваши задачи — обсудим по телефону или в мессенджере.
